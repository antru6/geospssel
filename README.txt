Queries, summaries and data from the experiment used for the Geospatial Source Selection paper submitted for ISWC2021.

queries.txt : the queries that we have used in the experiment
summaries/ : summary data used for each one of the federation engines of the experiment
data/ : adm9, invekos9, and snow9 (the relevant data that correspond to the smallest state of our experiment, i.e. the state of Vienna).
